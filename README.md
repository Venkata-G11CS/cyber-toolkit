# 🛡️ Cyber Toolkit

A collection of Python-based cybersecurity tools for network reconnaissance and enumeration.

---

## 🔧 Tools Included

| Tool | Description |
|------|-------------|
| `port_scanner.py` | Scans common ports on a target host |
| `ping_sweeper.py` | Sweeps a network range for live hosts |
| `banner_grabber.py` | Grabs service banners from open ports |

---

## ⚙️ Requirements

- Python 3.x
- Windows / Linux / Mac

No external libraries needed — uses Python built-ins only.

---

## 🚀 Usage

### Port Scanner
\```
python port_scanner.py
\```
Enter target host when prompted (e.g. `192.168.1.1`)

### Ping Sweeper
\```
python ping_sweeper.py
\```
Enter network base when prompted (e.g. `192.168.1`)

### Banner Grabber
\```
python banner_grabber.py
\```
Enter target IP when prompted
